# Banking-System-JAVA
Java Banking System project with account management
  
This project demonstrates basic banking operations like creating accounts, depositing money, withdrawing money, and checking balance using Java concepts such as classes, objects, ArrayList, and user input.

## Features
- Create a new bank account
- Deposit money
- Withdraw money
- Check account balance
- View account details
- Store multiple accounts using ArrayList

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- ArrayList
- Scanner (for user input)

## Project Structure
BankingSystem.java  
Contains the main program and the `Accounts` class which manages account details and banking operations.



## Learning Purpose
This project is useful for beginners learning:
- Java basics
- Object Oriented Programming
- Handling multiple objects
- Console based applications
